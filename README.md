# Leaflet JS MAP 

This project creates a map using LeafletJS and ReactJS libraries

## Installation

**src\components\Map.js**

Change LongLat here : 

```bash
 ...
       <MapContainer center={[21.239269, 105.703560]} zoom={15} style={{ height: "100vh", width: "100%" }}> 
 ...
```
